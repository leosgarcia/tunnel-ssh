wltech SSH Tunnel Manager
=========================

Versão: 1.2.0

Este pacote contém a versão Windows do aplicativo, pronta para uso.

Arquivos incluídos:
- wltech-tunnel.exe
- release-notes.md
- README.md

Instruções:
1. Execute wltech-tunnel.exe.
2. Se o seu sistema não tiver OpenSSH configurado, instale o cliente SSH do Windows ou Git for Windows.
3. Para suporte, consulte o repositório do projeto.
