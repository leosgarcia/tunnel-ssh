# Release 2.0.0

## O que há de novo
- Verificação automática de atualizações no GitHub oficial
- Confirmação obrigatória antes de qualquer download
- Notas da versão e progresso do download na interface
- Opção para ignorar o aviso de uma versão específica
- Validação de origem, tamanho, formato e SHA-256 do executável
- Instalação com backup e rollback automático em caso de falha
- Melhorias de alinhamento e proporção dos controles principais

## Arquivos de distribuição
- dist/wltech-tunnel.exe

## Requisitos
- Windows 10/11
- OpenSSH disponível no sistema ou via Git for Windows
